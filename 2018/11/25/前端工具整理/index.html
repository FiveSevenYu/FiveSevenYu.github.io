    书单推荐

    以下书单是本人已经购买的（大部分）按学习路径排序,从简单到精(fang)通(qi)

    HTML

    《Head First HTML与CSS》
    HTML，CSS最佳入门

    CSS

    《CSS权威指南》
    暂时好像只出到了CSS2.1版本的。

    《CSS揭秘》
    我买的没有魔法哥签名，要不我翻开给你看看？

    《CSS世界》
    张鑫旭大神对CSS的理解

    JavaScript

    《JavaScritp DOM编程艺术》
    JavaScript入门最佳食用方式

    《JavaScript高级程序设计》
    经久不衰“红宝书”

    《JavaScript权威指南》
    动物书系列-我买来当字典用的，名符其实枕头书，午睡的时候拿来垫头。作者:。。。。

    《JavaScript语言精粹》
    动物书系列-推荐啃完红宝书立即食用，以助消化

    《你不知道的JavaScript》
    全面介绍了JavaScript中常被人误解和忽视的知识点，分为上、中、下三卷。能力提高+1

    《JavaScript设计模式》
    动物书系列-室友大佬说等我工作一年之后再来看这本书，emmmm我不信，让他跟我举个列子讲一下，两个小时过去了，懵逼脸。

    喂！！没有了没有了，真的没有了。我桌子下的都拿出来了。
