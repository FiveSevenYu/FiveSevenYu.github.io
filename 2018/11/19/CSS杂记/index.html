    perspective 透视

    perspective属性

    定义：perspective属性定义3D元素距视图的距离，以像素计。当为元素定义perspective属性时，其子元素会获得透视效果。而不是元素本身。

    stransform-style

    指定嵌套元素如何在3D空间中呈现

    flat为默认值，所有子元素在2平面呈现，preserve-3d所有子元素在3d空间中呈现。
    3D变形中perspective()函数和perspective属性作用相同，但不同的地方是：
    perspective用在舞台元素上（变形元素们共同的父元素）
    perspective()用在当前元素上，并且可以与其他transform元素一起使用，并且取值只能大于0。

    perspective-origin

    用来决定perspective属性的源点角度。

    HTML的Page Visibility API

    目前关于页面的可见性API有两个属性，一个事件，如下：

    document.hidden:Boolean值,表示当前页面可见还是不可见

    document.visibilityState:返回当前页面的可见状态，取值有 hidden visible prerender preview

    visibilitychange:当前可见状态改变的时候触发事件
