    • vscode快捷键速查

    提高效率，珍爱生命！！！
    整理了一些比较常用的快捷建给小可爱们。

    基础编辑
    按键 功能
    Alt+ ↑ / ↓ 向上/向下移动行
    Shift+Alt + ↓ 向上/向下复制行
    Ctrl+Shift+K 删除行
    Ctrl+Enter 在下面插入行
    Ctrl+Shift+Enter 在上面插入行
    Ctrl+Shift+\ 跳到匹配的括号
    Ctrl+] / [ 缩进/缩进行
    Home 转到行首
    End 转到行尾
    Ctrl+Home 转到文件开头
    Ctrl+End 转到文件末尾
    Ctrl+Shift+[ 折叠（折叠）区域
    Ctrl+Shift+] 展开（未折叠）区域
    光标定位与选择
    按键 功能
    Ctrl + Alt +↑/↓ 在上/下插入光标
    Ctrl + U 撤消上一个光标操作
    Shift + Alt + I 在选定的每一行的末尾插入光标
    Ctrl + I 选择当前行
    编辑器管理
    按键 功能
    Ctrl+F4, Ctrl+W 关闭编辑器
    Ctrl+\ 拆分编辑器
    Ctrl+ 1 / 2 / 3 聚焦到第1，第2或第3编辑器组
