    服务器系统的选择

    Linux(centos,Ubuntu) //稳定
    windows(widowsSever) //操作简单

    购买:

    设置🔗连接密码
    设置80端口

    管理:

    使用Gitbash连接服务器
    使用ssh user@remote_ip连接到远程服务器

    web服务器的选择

    Nginx: 高性能,高并发
    Apache: 多平台,安全,流行
    Tomcat: 多用于java做jsp解析

    安装nginx

    yum install nginx
    输入 nginx -v来查看是否安装成功

    配置web服务器

    更改部分：
    1.user nginx 为 user root
    2.http下的location改为
    root /root/www/
    index index.html index.htm

    :wq保存并退出

    cd /root/www/ 进入www文件夹(如果没有该文件则在nginx文件夹下先进入root文件夹,再mkdir www来新建www文件夹,再进入该文件夹)

    上传网站到服务器

    打开gitbash进入本地网站文件夹的根目录

    如果没有,需要安装

    连接服务器

    ssh user@remote_ip
    进入www文件夹查看文件是否上传成功

    ps -ef | grep nginx
    查看当前线程
    nginx -s stop //关闭server服务

    nginx -t 查看当前状态
    如果不成功查看nginx.conf配置是否错误,比如少了;

    nginx -s reload 重新加载

    现在只要80端口配置正常,应该就能通过ip访问服务器的网站了.

    添加域名

    在域名控制台中配置解析对应ip即可
