    正则表达式是什么？

    正则表达式(Regular Expression) 简写:RegExp

    正则表达式是对字符串操作的一种逻辑公式，
    就是用事先定义好的一些特定字符、以及这些特定字符的组合，组成一个新的“规则字符串”，
    这个“规则字符串”就是用来表达对字符串的一种过滤逻辑。

    正则表达式如何创建？

    1.字面量法

    var reg1 = //;

    2.利用构造函数创建

    var reg2 = new RegExp();

    元字符/预定义类

    . [^\n\r] 除了换行和回车之外的任意字符
    \d [0-9] 数字字符
    \D [^0-9] 非数字字符
    \s [\t\n\r\f\v] 不可见字符
    \S [^\t\n\r\f\v] 可见字符
    \w [a-zA-Z0-9] 单词字符（所有的字符数字和_）
    \W [^a-zA-Z0-9] 非单词字符

    正则的参数

    i 忽略大小写
    g 全局匹配 global
    gi 全局匹配+忽略大小写

    自定义类

    /heima/.test(“123d”); test检测的字符串里面有没有包含heima这个字符，如果有就是true。
    这里面的heima必须完整的出现一次，test方法检测才是true。

    或和优先级 |

    /he(i|m)a/.test(“hei”); 解析出来也就是/heia | hema/

    简单类 []

    简单类只代表一个字符
    如果用test检测的字符串中，出现[]中的字符任意一个，就是true;
    /[abc]/.test(“a”); true

    边界

    /^ / 会匹配行或者字符串的起始位置
    /^\d/.test(“1sasakb”); true 如果检测的字符串的起始位置是数字就是true。

    /$/ 会匹配行或字符串的结尾位置
    /ac$/.test(777ac); true

    ^$组合在一起，就表示严格匹配 只能出现一次 有且只有一次
    /^abc$/.test(“abc”); true

    负向类 [^ ]

    如果用test检测的字符串中，有 除了中括号中的内容之外的任意一个，出现一次或者一次以上，就是true。
    /[^abc]/.test(‘abc1’); true

    范围类

    组合类

    量词

    “*” 出现零次或更多 x>=0 {0,}
    “+” 出现一次或者更多次 x>=1 {1,}
    “?” 出现零次或者一次 x=(0||1) {0,1}

    {n} n次 x=n
    {n,} 出现n次或者更多 x>=n
    {n,m} 出现的次数在n次和m次之间（包含） n<= x <=m

    案例

    1.判断是否存在中文

    在js中，不管什么字符都有一个对应的编码，中文对应的编码 [\u4e00-\u9fa5]
    /[\u4e00-\u9fa5]/.test(“asdfkiasdGksd123一”); true

    2.验证姓名

    思路：名字由中文开始和结束，长度2位或者4位
    /^[\u4e00-\u9fa5][2,4]$/.test(“畅畅”);

    3.验证邮箱

    邮箱的格式xxx@xxx.xx
    /^\w+@\w+(.\w+)+$/.test(‘mail’) 严格匹配 单词字符

    字符串方法

    replace()
    利用replace和正则来替换字符串中的数字
    var str = “100合适或200深刻的5023”
    var strNew = str.replace(/\d+/g,’h’);

    strim()
    去掉字符串前后的空格
    需求：用正则表达式写一个strim方法
    思路：把str中的不可见字符替换成空字符串

    字符串方法 search检索字符串

    之前学过的检索字符串的方法
    str.indexOf(‘d’) 返回该字符的下标，如果没有返回-1
    str.lastIndexOf(‘d’) 从右往左检索，返回字符的下标，没有则返回-1

    匹配-search()
    返回与正则相匹配的字符的下标
    需求：检索出第一次出现数字的下标
    var index = str.search(/\d/);
    console.log(index); //1

    分割-split
    我们之前只能用一些固定的字符比如，|来实现分割
    var str = “张三|李四|王二麻子”;
    var str1 = “张三100李四200王二麻子”;
    var arr = str.split(“|”);
    console.log(arr); // [“张三”, “李四”, “王二麻子”]
    需求：我们需要分割str1,它们中间的分隔符是数字，这时便需要用到正则
    var arr2 = str1.split(/\d+/);
    console.log(arr2); // [“张三”, “李四”, “王二麻子”] 把正则作为一个分隔符

    正则对象相关的方法

    test()方法
    语法：正则对象.test(需要检测的字符串);
    返回值：如果检测的字符串满足这个正则对象那就是true，不满足就是false。

    exec(str) 提取满足正则reg规则的第一个，以数组的形式返回
